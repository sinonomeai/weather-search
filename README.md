# 项目说明

## 技术栈

### 前端框架

- **React**
- **TypeScript**
- **Vite**

### UI与样式

- **Tailwind CSS **
- **Ant Design**
- **qweather-icons(和风天气图标)**

### 状态管理与路由

- **Zustand**：
- **React Router DOM**

### 数据可视化

- **Chart.js 4.5.1**
- **react-chartjs-2 5.3.1**

### 工具库

- **pinyin-pro 3.28.0**

### 后端模拟

- **JSON Server**

## 使用
```
bash
npm run server
pnpm dev
```



## 核心功能

### 已实现功能

+ 用户登录/注册（JSON Server模拟后端）
+ 城市天气查询（和风天气API）
+ 实时天气显示
+ 7天天气预报
+ 24小时逐小时预报
+ 生活指数建议
+ 城市收藏（添加/删除/查看）
+ 加载骨架屏及loading动画
+ 刷新按钮
+ 本地保存用户登录状态
+ 路由守卫管理员页面（非管理员无法进入）------还需了解react底层的页面渲染以及数据更新
